# 📂 docs/ — Tài liệu dự án theo VERSION

Mỗi version của app = 1 folder (`v1.3.0`, `v1.4.0`, …). Trong mỗi version, tài liệu chia theo
**5 vai trò** — mỗi vai trò một folder, là nơi vai trò đó bỏ tài liệu / file của mình.

| Folder        | Vai trò |
|---|---|
| `po_docs/`    | **PO** — tài liệu tính năng (mô tả dự án, nguồn cho các vai trò khác đọc) |
| `tester/`     | **Tester** — test case |
| `dev_mobile/` | **Dev (mobile)** |
| `be_docs/`    | **BE** |
| `ai_docs/`    | **AI** |

- **Version đang hoạt động: `v1.4.0`** — docs mới rơi vào version hiện tại.
- Nhận diện vai trò + phân quyền + version đang chạy do `brain/` quản (đang hoàn thiện).
